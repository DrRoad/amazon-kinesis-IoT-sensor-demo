AWS IoT Sensor Demo - Producer

You sensor data 'Screen-Adjusted DeviceOrientation' and 'Screen-Adjusted DeviceMotion', and your device type and Cognito Id are tracked every 300ms. The data is send every seconds via https to the AWS Cloud - region and stored for data anlyses. Have fun getting part of an Internet of Things Sensor Demo.
